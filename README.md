# LionMinestom
A basic minestom project designed to have a lightweight lobby integrated in LionAPI
> <strong>The features provided should be similar to SimpleLobby. Look [here](https://github.com/LionCraft08/SimpleLobby) for more information.</strong>

# Setup
Download the jar and execute it once to create the config file. Now, complete the configuration. If you have questions regarding the TCP connection to [LionVelocity](https://github.com/LionCraft08/LionVelocity), look at the config.yml file of [LionAPI](https://github.com/LionCraft08/LionAPI)
